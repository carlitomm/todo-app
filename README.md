# Full-Stack Todo App  
### React + Vite + Redux Toolkit + TailwindCSS + Node.js + Express + TypeScript

The project is split into two folders:

/todo-app-frontend
/todo-app-backend

# 1. Backend Setup (Node.js + Express + TypeScript)

### Folder:
`todo-app-backend`

## Installation

```bash
cd todo-app-backend
npm install
```

## Run Backend
```bash
npm run dev
```

#2. Frontend Setup (Vite + React + TS + Tailwind + Redux Toolkit)

## Installation
```bash
cd todo-app-frontend
npm install
```

## Run Frontend

```bash
npm run dev
```

the app will be available at `http://localhost:5173`
